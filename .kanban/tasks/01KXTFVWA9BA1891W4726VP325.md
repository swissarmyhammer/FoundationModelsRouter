`CompactionEvaluation: Evaluation`:
- **Subject**: compact the sample's transcript with the prompt under test, resume a session over the result, ask the sample's question — return answer + `CompactionResult`.
- **Dataset**: 20–30 hand-written seed transcripts (`ArrayLoader<PlantedFacts>`) with varied lengths, tool traffic, and multiple planted facts in the transcript head ("the API key lives in `.env.example`", "we chose tabs over spaces"); questions answerable only from folded content.
